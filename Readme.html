Installing the Engineering Vibration Toolbox for Octave

Joseph C. Slater

April 15, 2014

1 Installation on a Windows machine

  1. There are two options. If you don’t know which to choose, choose the first.
    1. If you do not have Cygwin installed, download octave from sourceforge at
      http://sourceforge.net/project/showfiles.php?group_id=2888
      This link may change, so you may have to go to
      http://www.gnu.org/software/octave/download.html
      to find it.
    2. If you do have Cygwin installed, go to http://www.cygwin.com/ and look for the octave package. The directories listed below modified to match the location of your Cygwin root directory.
  2. Put the unzipped ovtoolbox folder in
    C:\Program Files\GNU Octave #\usr\share\octave\site\m where # is the version number of your Octave installation.
  3. Replace the file C:\Program Files\GNU Octave \#\usr\share\octave\site\m\startup with the file startup in the ovtoolbox directory. This will make Octave behave more Matlab like and help assure greater compatibility.
  4. Type “help vtb1” after running Octave to see if the installation of the Engineering Vibration Toolbox worked.
  5. Create a directory
    C:\Program Files\GNU Octave #\usr\share\octave\site\m\myfiles where you can put your script and program files.

2 Installation on a Unix machine

Before the Engineering Vibration Toolbox for Octave can be installed, Octave, from http://www.octave.org must be installed. Please consult that website for installation instructions.

The toolbox can go in any directory as long as the Octave path to it is set properly in the .octaverc file in the user’s home directory. Like any of the other toolboxes, you should not save personal files inside the ovtoolbox directory since you may inadvertently lose them when you update (i.e. you may decide to delete the directory and replace everything).

Example: On my mac (which is BSD unix), my .octaverc file contains the command: path(path,”/Volumes/LaCie/Documents/MyMath/ovtoolbox”);

3 Using the Engineering Vibration Toolbox

Typing ’help vtb?’ will provide a table of contents for the files related to chapter ’?’. Typing ’help codename’ will provide help on the particular code. Note that the ’filename’ is ’codename.m’.

Engineering Vibration Toolbox commands can be run by typing them with the necessary arguments just as any other Octave/MATLAB commands/functions. For instance, vtb1_1 can be run by typing ”vtb1_1(1,.1,1,1,0,10)”. Many functions have multiple forms of input. The help for each function shows this flexibility.

4 Printing plots

To print a plot, the easiest way may be to type “print” from the Octave console. This should bring up dialog boxes that enable you to send the current plot to your current Windows printer.

The “print” command can also be used to generate graphics files for embedding into your favorite word processing program. Type “help print” for details. By default the file will end up in the directory
C:\Program Files\GNU Octave #\octave_files
